# Changelog

## [1.0.0] - Initial release

### Added

- WooCommerce product, stock, and pricing synchronization from Victoria ERP.
- Promotion engine with cart and checkout rule support.
- Product catalog import and manual sync from ERP.
- Admin settings page and ERP connection testing.
- Action Scheduler and WP Cron fallback for recurring sync jobs.
- REST API endpoints for ERP webhooks and plugin status.
- Logging, caching, and uninstall cleanup.
