# woocommerce-victoria-erp

This repository contains the `Victoria ERP Connector` WordPress plugin for WooCommerce.

The plugin integrates WooCommerce with Victoria ERP to synchronize stock, pricing, products, and promotions.

## Package contents

- `victoria-erp-connector/`: WordPress plugin source files
- `victoria-erp-connector/victoria-erp-connector.php`: plugin bootstrap and autoloader
- `victoria-erp-connector/includes/`: plugin PHP classes and subsystems
- `victoria-erp-connector/templates/`: public-facing templates and My Account pages

## Release

This repository is prepared for the `v1.0.0` release.
